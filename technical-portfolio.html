Cave Exploration

In this project I implemented a single-threaded (soon to be multi-threaded) cave exploration tool using C++ and various Windows API functions.

January 2024

The goal of this project is to showcase various graph traversal algorithms by exploring threading and pipes by using the Windows API to talk to the kernel. To accomplish this task, a rover known as the command center must send out a number of flybots to find the exit. To make this more realistic, each vertices within the weighted graph, or cave in the cave system, has a specific light intensity. The closer you are to the exit of the cave system, the greater the light intensity. Users can select either BFS, DFS, A*, or a modified version of BFS prioritizing light intensity to traverse the cave system.

Data Structures and Algorithms Implementation

This project is a culmination and implementation of the most important topics in Data Structures and Algorithms.

August 2023

The C++ standard template library is an extensive library providing users with an efficient and powerful abstraction of popular data structures at their fingertips, so why not replicate it? This project showcases my growth as a Computer Science student as I navigated through one of the most notorious and fundamental courses. I implemented unordered_map (hashmap with chaining), vector, list (linked list), and priority_queue. All implementations incorporate copy constructor, move constructor, destructor, copy assignment operator, and move assignment operator to follow the rule of five. Each of the data structures include their respective implementations of popular algorithms, along with custom iterators.

Optimized Fibonacci - Multiple Precision Addition

This project demonstrates how a simple algorithm can be tweaked to enhance performance by over 1000%.

November 2023

The standard recursive approach for calculating a fibonacci number is far from efficient and uses a considerable amount of stack space due to the amount of function calls. Furthermore, large numerical calculations in C++ are heavily limited by the range of standard datatypes. This project modifies a given multi-precision addition algorithm to handle large calculations from an iterative fibonnaci calculation. To optimize further, a portion of the multi-precision addition algorithm is written in x86-64 Assembly utilizing carry propogation and loop unrolling.
